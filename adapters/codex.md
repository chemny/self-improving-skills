# Codex Adapter

Codex can use this skill when its description triggers from the user's request.

## Best usage

- Use the skill for explicit memory requests, task reflection, repeated errors, and rule promotion.
- Read the workspace's agent instruction file, such as `AGENTS.md`, before changing workspace behavior when that file exists.
- Use persistent files only when the user asked to remember or when the learning is clearly durable.

## Common prompts

```text
记住我的写作风格是...
总结这次任务里值得沉淀的经验。
把这个错误记录下来，之后别再犯。
判断这条规则是否应该写入当前项目的 agent instruction file。
帮我清理一下过期的 agent 规则。
```
